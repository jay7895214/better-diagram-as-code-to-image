Better Diagrams as Code to Image | 程式碼轉圖表工具

A powerful tool to render Mermaid and PlantUML code in real-time and export them as high-quality images.

將 Mermaid 與 PlantUML 程式碼即時渲染並匯出為高品質圖片的強大線上工具。支援行內選色、高解析度匯出與一鍵分享功能。

Features / 功能特色

  • Real-time rendering for Mermaid & PlantUML / 即時渲染 Mermaid 與 PlantUML
  • Inline Color Picker / 智慧行內調色盤，直接點擊色塊切換顏色
  • High-quality PNG, JPG, SVG export / 支援匯出最高 20 倍解析度的 PNG、JPG 與 SVG
  • Shareable URL / 可分享的 URL 連結
  • Dark Mode Support / 支援深淺色模式
